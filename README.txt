# Restaurant Recommender

A website that will help users decide what restaurants/foods they would like to go to using options and minigames to help decide.

# Initial Features

- Filtering using predetermined options (location, budget, cuisine)
- Use different options to choose the restaurant (choices, tournament mode, wheel)
